Todas las pruebas fueron ejecutadas en Python version 3.11.4

Instrucciones y consideraciones para usar el programa:

    -> Los archivos tienen los nombres problema1.py el cual contiene el problema de "Fuerza bruta recusiva" y problema2.py el cual contiene el problema de "Programacion Dinamica".
    -> Se asume que los .dat se encuentran dentro del mismo directorio donde estan los .py.
    -> Abrir una terminal, acceder la carpeta donde se encuentra los archivos .py.
       Si esta en una terminal de Windows ejecutar el codigo: "type <Nombre del input>.dat | python <Nombre del problema a probar>.py".
       Si esta en una terminal de Linux ejecutar el codigo: "python3 problema1.py < input.dat".
    
Supuestos:
     * Se considero que los Problemas debian ir en archivos separados, los cuales tienen por nombres problema1.py y problema2.py.
     * Se considero que se utilizaran casos que no rompan el proposito de la tarea como tal.
